# Batch 6: 消息性能指标增强 — 测试计划

## T1: 指标显示验证

- [ ] 发送消息后，Agent 回复的元信息行显示 TTFT
- [ ] 元信息行显示 TTLT
- [ ] 元信息行显示 Total Duration
- [ ] 现有 Response time 仍正常显示

## T2: 格式验证

- [ ] <1000ms 的值显示为 ms（如 `324ms`）
- [ ] ≥1000ms 的值显示为 s（如 `1.2s`）

## T3: 多消息场景

- [ ] 连续发送多条消息，每条的指标独立且正确
- [ ] 切换 Agent 后指标仍正常
- [ ] 新建对话后指标正常

## T4: 流式 vs 非流式

- [ ] 非流式消息（单条回复）指标正确
- [ ] 流式消息（多个 activity 组成）TTLT 准确反映最后一个 token

## T5: 回归测试

- [ ] AI Companion 面板正常
- [ ] 语音功能正常
- [ ] 消息渲染样式无变化
