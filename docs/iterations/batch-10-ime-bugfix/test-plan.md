# Batch 10: IME 回车误触发送修复 — 测试计划

## T1: 中文 IME

- [ ] 输入拼音并按 Enter 选词，不发送消息
- [ ] 选词完成后再次 Enter 才发送

## T2: 日文 IME

- [ ] 假名转换时 Enter 不触发发送
- [ ] 转换完成后的 Enter 可正常发送

## T3: 英文输入

- [ ] 非 IME 状态 Enter 正常发送

## T4: 换行行为

- [ ] Shift+Enter 仍可换行（如当前支持）

## T5: 稳定性

- [ ] 快速输入与切换输入法时无误触发送
- [ ] 不引入重复发送或事件丢失
