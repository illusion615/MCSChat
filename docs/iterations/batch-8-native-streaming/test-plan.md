# Batch 8: Copilot Studio 原生 Streaming 输出 — 测试计划

## T1: 能力确认

- [ ] 能明确判断当前环境是否支持原生 streaming
- [ ] 结论可复现（同配置重复验证一致）

## T2: 消息完整性

- [ ] 原生流式模式下消息无丢字或乱序
- [ ] 分片结束后消息状态能正确收口

## T3: 回退可用性

- [ ] 关闭原生 streaming 后恢复模拟流式
- [ ] 回退后消息链路与现网一致

## T4: 性能与指标

- [ ] TTFT/TTLT/Duration 指标仍可计算
- [ ] 首段到达延迟优于或不劣于现有方案

## T5: 稳定性

- [ ] 弱网下无异常堆积或重复渲染
- [ ] 连接重连后 streaming 行为正常

## T6: 兼容性

- [ ] 普通非流式消息仍可正常显示
- [ ] Adaptive Card 与系统事件不受影响
