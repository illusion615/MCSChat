# Batch 11: Streaming 样式与速度优化 — 测试计划

## T1: 样式可用性

- [ ] 至少 3 种样式可切换
- [ ] 样式切换后新消息显示正确

## T2: 速度控制

- [ ] 速度设置能明显改变节奏
- [ ] 各档位无异常跳字或卡顿

## T3: 持久化

- [ ] 刷新页面后样式与速度配置保留

## T4: 兼容性

- [ ] 非流式消息显示不受影响
- [ ] Adaptive Card 渲染不受影响

## T5: 性能

- [ ] 长消息场景无明显卡顿
- [ ] 连续消息场景无渲染堆积

## T6: 兜底能力

- [ ] 任一策略异常时可回退 instant
- [ ] 回退后消息仍完整可读
