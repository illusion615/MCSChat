# Batch 11: Streaming 样式与速度优化 — 需求文档

## 目标

改进消息流式输出的视觉效果，提供多种 streaming 样式和速度供用户在设置界面中选择。

## 当前问题

当前 streaming 方式（逐字显示）不够优雅，视觉体验生硬。

## 需求

### 多种 Streaming 样式

| 样式 | 效果描述 |
|------|----------|
| **逐字（Typewriter）** | 当前方式，逐字符出现 |
| **逐词（Word by word）** | 按单词/词组出现，更自然 |
| **逐句（Sentence）** | 按句子出现，适合快速阅读 |
| **淡入（Fade in）** | 文本以淡入动画分段出现 |
| **即时（Instant）** | 无动画，直接显示完整消息 |

### 速度控制

- 提供速度滑块或档位选择（慢/正常/快/极快）
- 不同样式有各自合理的默认速度

### 设置界面

- 在设置面板中新增 "Streaming" 或 "Message Display" 区域
- 样式选择（下拉/单选）
- 速度调节（滑块）
- 实时预览效果

## 验收标准

1. 至少 3 种 streaming 样式可选
2. 速度可调节
3. 设置面板中有对应的配置 UI
4. 选择后即时应用
5. 设置持久化到 localStorage
