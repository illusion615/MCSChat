# Batch 12: Appearance 面板移至右侧栏 — 需求文档

## 目标

将 Appearance（外观设置）从当前位置移至右侧栏，以 icon 按钮形式放置到左侧 command bar，修改后即时应用到界面。

## 需求

### 布局变更

1. **左侧 Command Bar**：新增一个 Appearance 图标按钮
2. **右侧栏**：点击该按钮后在右侧打开 Appearance 面板（类似 AI Companion 面板的位置）
3. **关闭**：再次点击 icon 或面板内关闭按钮关闭面板

### 即时应用

- 修改任何外观设置后即时应用到当前界面
- 无需刷新页面
- 包括：主题颜色、消息图标显示/隐藏、用户图标设置等

### 面板内容

保持现有 Appearance 设置项不变，仅改变其展示位置：
- 主题/颜色设置
- 消息图标开关
- 用户图标选择

## 验收标准

1. 左侧 command bar 有 Appearance 图标
2. 点击后右侧面板打开
3. 修改设置即时生效
4. 面板可关闭
5. 不影响其他右侧面板（AI Companion 等）的正常使用
